# nem2-scenarios

Catapult behavior validation using scenarios.

## Requirements

* Node.js
* npm

## Installation

``npm install``

## Runt tests

``npm test``

## Write an scenario

* [Gherkin syntax](https://docs.cucumber.io/gherkin/)
